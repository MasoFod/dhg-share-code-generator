# 漆黑猎场物品分享代码生成器
[使用地址](https://masofod.github.io/dhg-share-code-generator/release/)
适用游戏版本：v1.6.1
本工具目前仅支持中文。
This tool currently only supports Chinese.

## 简介
本工具可以直接选择物品类型、名称、稀有度和词条等信息直接生成物品分享代码。

## 注意
1. 数值可以随意填写，且本工具不对代码可用性保证。若遇问题，请先确认自己设置的属性是否正常；若确定一切正常却仍无法使用，请发issue，给出完整截图。本工具不保证后续更新。
2. 部分名称与游戏内不同，实际效果相同。
3. 词条顺序基本与游戏内一致，请对照使用。
4. 数值项须填整数，百分比项须填百分数实际对应的小数，负数项须填负数。

## 关于源码
本项目使用Godot4.5.1制作。
